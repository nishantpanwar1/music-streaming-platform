# Music Streaming Platform
## Overview
The music streaming platform is a web application that allows users to stream music.
## Key Components
1. **Music Library**: Integrate a music library like YouTube Music.
2. **Recommendation System**: Implement a robust recommendation system using collaborative filtering and content-based filtering.
3. **User Authentication**: Implement user authentication using email or phone.
## Technology Stack
1. **Backend**: Java 17 (Spring Boot) + microservices
2. **Frontend**: React (Node.js)
3. **Database**: PostgreSQL + Redis for caching
4. **Cloud**: AWS
